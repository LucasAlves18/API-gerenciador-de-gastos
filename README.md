# Gerenciador de gastos

- A ideia desse projeto é criar um CRUD com autenticação JWT, Fast Api e SQLAlchemy.
- O projeto foi separado em camadas, e para rodar ele basta configurar seu interpretador e baixar as depêndencias, e startar o projeto
- Db é criado automaticamente ao startar o projeto com a função nativa do SQLAlchemy

## Como rodar
pip install -r requirements.txt
python main.py
